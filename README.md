# A mentorship web application
The goal of this project is to connect like minded people to each other under one mentor. These mentorship programs apply to all areas of life.

## How to use
1. open the project file path in cmd if you are on windows
2. run this command `python manage.py runserver`
3. copy the url that will be diplayed in the response and paste it on any web browser
* what to look for.
* ```Starting development server at http://path_for_url/```
4. Create an account by registering
